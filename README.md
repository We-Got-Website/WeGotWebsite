## ✨ Features

- **🎯 Precision Focused** - Tailored designs that align perfectly with your brand identity.
- **⚡ Blazing Fast** - Built on Vite 6 for near-instant load times and smooth interactions.
- **📱 Ultra Responsive** - Seamless experience across all devices, from mobile to ultra-wide monitors.
- **💫 Fluid Animations** - Powered by `motion` for a premium, interactive feel.
- **🛠️ Service Oriented** - Dedicated pages for About, Careers, Security, and more.
- **📝 Lead Generation** - Built-in Contact & Project inquiry forms.

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
- **Routing:** React Router 7
- **Build Tool:** Vite 6

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/WeGotWebsites.git
   cd WeGotWebsites
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:3000`.
